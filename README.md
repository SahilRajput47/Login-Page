## 🧰 Setup Instructions

1. Clone this repository:
   ```bash
   git clone https://github.com/SahilRajput47/Login-page.git
   ```
2. Navigate to the login form folder:
   ```bash
   cd Login-page/login\ form
   ```
3. Open `index.html` in your browser.
